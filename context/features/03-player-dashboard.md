# Player Dashboard

## Overview

The dashboard is the landing page after login, showing all games the player belongs to with at-a-glance activity information.

## Requirements

### Game List
- Landing page after login shows all games the player belongs to
- Clicking a game navigates to the game view

### Game Cards
- Each game card shows: game name, player's primary character name (linked to character sheet), number of active scenes, and a "new activity" indicator if anything changed since last login
- Players with multiple characters see their primary character name with a "+N" count for additional characters

### Membership Status
- Removed players see their game with a "Former" indicator
- Banned players do not see the game at all
